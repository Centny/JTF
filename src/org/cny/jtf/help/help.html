Instruction:
 the ScorpiontTestFixture be used to import the data to test suite in the
 EJB3Unit through the .xls file and export the data to .xls file from the database.


User Guide:
 1.how to use the test base class:see the org.scorpion.testbase.ScorpionTestFixtureTest.java in STF*.jar
 2.how to use the export data tools:
  1).extract the STF*.zip to the folder,like "STF"
  2).cd ~/STF
  3).java -jar STF*.jar -pf ~/profile.properties -xf ~/db.xls
 the sample profile configure file is in the help folder "profile.proeprties"
